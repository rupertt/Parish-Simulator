# Game Design

## Current Goal

Build a first playable local multiplayer 2D pixel-art prototype.

## Prototype Features

- One shared room.
- Multiple players connected from browsers on the same local network.
- WASD movement.
- Simple colored square placeholders.
- Movement constrained to room boundaries.

## Out of Scope for Now

- Authentication
- Cloud hosting
- Matchmaking
- Database storage
- Inventory
- Combat
- Advanced game systems
