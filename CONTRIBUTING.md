# Contributing

Keep changes small and focused. This project is currently a first playable local multiplayer prototype.

## Before Opening a Pull Request

```bash
npm install
npm run build
```

## Guidelines

- Do not add authentication, matchmaking, cloud hosting, databases, inventory, combat, or advanced systems yet.
- Prefer simple TypeScript types shared between client and server.
- Keep multiplayer events easy to inspect and debug.
- Update documentation when setup or controls change.
